# Sistema de Banco de Dados - Controle de Vendas e Estoque

Este repositório contém o modelo físico e lógico de um banco de dados para controle de vendas, clientes, funcionários, fornecedores e gerenciamento de estoque/reabastecimento.

## 📌 Diagrama Entidade-Relacionamento (DER)
![Diagrama do Banco de Dados](diagrama-der.jpg)

## 🛠️ Como usar
Basta copiar o conteúdo do arquivo `script.sql` e executar no seu gerenciador de banco de dados preferido (MySQL, PostgreSQL, etc.).
